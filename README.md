# Define: A CLI for Dictionaries

Look up words in the dictionary directly from your terminal.

## Installation

## Example

`>>> define meaning`
`>>> meaning (noun)`
`>>> `
`>>> mean*ing | ˈmē-niŋ`
`>>> `
`>>> the thing one intends to convey especially by language : purport`
